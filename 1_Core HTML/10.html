1. HTML

    Primary Use Cases Key Features/Benefits
    Building the structure of web pages
    • Defines the structure of web content.
    • Uses tags for elements like headings, paragraphs, and links.
    • Provides semantic elements for better meaning and accessibility.

  2. CSS

    Primary Use Cases Key Features/Benefits
    Styling and layout of web pages
    • Controls the presentation and style of web pages.
    • Separates content from design
    • Enables responsive and consistent layouts.

  3. JavaScript

    Primary Use Cases Key Features/Benefits
    Adds interactivity to web pages.
    • Can manipulate and modify web page content in real-time
    • Dynamically updates content without page reload.
    • Supports asynchronous operations and works on both client and server sides.
